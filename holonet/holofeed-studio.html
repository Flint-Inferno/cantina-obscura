HOLONEWS HEADLINES
Click a headline to expand/edit it. Each story has a headline, a body paragraph, and a tag.
Tags are news outlets — define and manage them in the ⚙ OUTLETS tab.
Stories cycle automatically — order here doesn't matter, the feed shuffles them.
NEWS OUTLETS
Define the news outlets that appear as badges on headlines. Each outlet needs a display name, a short key (used internally in the HEADLINES data), and a color.
The tag picker on each headline updates automatically when you add or rename an outlet.
SPONSORED ADS
Ads appear roughly once every 3 headlines in the feed, marked with an amber SPONSORED badge.
The Cantina Obscura ad contains the free-drink easter egg — keep it if you want that to work.
PODRACER NAME POOL
Each race picks 6 names at random from this pool.
You need at least 6 names. More names = more variety between races.
EXCHANGE LISTINGS
Each stock shows a live scrolling line graph on the Exchange tab.
Symbol: short all-caps ticker (e.g. KDY).   Base price: starting credit value.   Volatility: price swing per tick — 0.01 = stable, 0.05 = very jumpy.
TIMING & SPEED SETTINGS
These control how fast things happen. All values are in seconds unless noted.
